Elements, Tags, And Attributes

Inline and Block Elements



Inline: It only takes necessary width.
Block: Takes up the full width.

List Of Some Inline Elements
<span>...</span> A generic inline container for text or other inline elements.
<a>...</a> Defines a hyperlink, which is used to link from one page to another.
<strong>...</strong> Indicates that the text is of strong importance, typically rendered in bold.
<em>...</em> Indicates emphasized text, typically rendered in italics.
<b>...</b> Renders text in bold without implying any extra importance.Renders text in bold without implying any extra importance.
<i>...</i> Renders text in italics without implying any extra emphasis
<code>...</code> Displays a single line of code or a code snippet.
<small>...</small> Renders text in a smaller font size.
<sub>...</sub> Renders text as subscript.
<sup>...</sup> Renders text as superscript.
<q>...</q> Represents a short inline quotation.
